Written by Frost

# Blogs       From now on, we'll be using this site to host any large announcements and updates we may have, as well as standard blog posts. If there's anything you want to see us cover, or questions you want answered, you can either contact us directly at qualitymuncher@gmail.com or send us a message in the [Discord server](https://discord.qualitymuncher.lgbt). # Quality Muncher       Currently, my primary goal for Quality Muncher is to add support for multi-queue, configuration files, better CLI usage, and more. Since this requires rewriting large sections of code, this will take some time. ### Cross-Platform Support       Given that Quality Muncher is a batch script, this version will likely never be cross-platform. However, Atzur has been making a [Python version of Quality Muncher](https://github.com/qm-org/qm-py). This version has all of the same basic features of the batch version of Quality Muncher, along with some of its more advanced features.
