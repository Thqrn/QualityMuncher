Quality Muncher

Next Level Compression

Quality Muncher is a simple program made to be the best at making the worst videos. It's extremely customizable, with tons of features, and is easy to install and use.

Installation

The Batch version is a Windows-only script made to be used with Send-To, which is fairly easy-to-use with a text-based GUI. The Go version is a cross-platform CLI utility. This is recommended only for users familiar with CLIs - or those who are willing to learn!
